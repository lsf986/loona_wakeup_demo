#### 8.1.3 关键算法实现

**1. EnergyVAD (能量+过零率VAD)**

```python
class EnergyVAD:
    def is_speech(self, frame: np.ndarray) -> bool:
        # 计算RMS能量
        rms = float(np.sqrt(np.mean(f * f) + 1e-9))
        # 计算过零率
        zc = float(np.mean(np.abs(np.diff(np.signbit(f).astype(np.int8)))))
        # 动态阈值判断
        thresh = self.noise_floor * (10 ** (self.energy_ratio_db / 20.0))
        # 能量高于阈值 且 过零率在语音区间
        return (rms > thresh) and (0.005 < zc < 0.40)
```

**实现原理：**
- 自适应跟踪噪声底，仅在非语音帧更新
- 能量阈值随噪声底动态调整
- 过零率用于区分语音(0.005~0.4)与高频噪声/低频噪声

**2. SpeakerTracker (声纹聚类)**

```python
def _flush_utterance(self) -> int | None:
    # 计算当前语音句的pitch/centroid中位数
    p_med = float(np.median(self._buf_pitch))
    c_med = float(np.median(self._buf_cent))
    
    # 与历史说话人对比
    for r in self._roster:
        semitones = abs(12.0 * np.log2(p_med / r["pitch"]))
        cent_d = abs(c_med - r["cent"])
        if semitones < self.PITCH_TOL_ST and cent_d < self.CENT_TOL_HZ:
            # 匹配成功，更新模板
            return r["id"]
    
    # 未匹配，创建新说话人
    return self._next_id
```

**实现原理：**
- 使用pitch(基频)和spectral_centroid(频谱重心)作为声纹特征
- 按句子(voiced帧连续)为单位聚类
- EMA更新模板适应音调漂移
- 区分说话人数用于"多人场景"判定

**1. GCC-PHAT 声源定位 (DOA)**

```python
def gcc_phat(sig: np.ndarray, ref: np.ndarray, sr: int = SR, 
             max_tau: float | None = None, interp: int = 4) -> tuple[float, float]:
    # FFT变换
    SIG = np.fft.rfft(sig, n=n)
    REF = np.fft.rfft(ref, n=n)
    
    # 广义互相关-相位变换
    R = SIG * np.conj(REF)
    R /= (np.abs(R) + 1e-12)  # 白化
    
    # 反变换到时域
    cc = np.fft.irfft(R, n=n * interp)
    
    # 寻找峰值(时延)
    peak = int(np.argmax(np.abs(cc))) - max_shift
    tau = float(peak) / (interp * sr)
    
    # 转换为角度
    ratio = max(-1.0, min(1.0, 343.0 * tau / mic_spacing))
    doa = float(np.degrees(np.arcsin(ratio)))
    return doa
```

**实现原理：**
- PHAT(相位变换)对白化处理，增强对混响的鲁棒性
- 4倍插值提高时延估计精度
- 利用双麦克风间距与声速，将时延转换为角度
- 默认麦克风间距40mm，角度范围±30°

**2. 浊音检测 (自相关法)**

```python
def autocorr_pitch(x: np.ndarray, sr: int = SR, 
                   fmin: float = 80.0, fmax: float = 400.0) -> tuple[float, float]:
    # 去均值归一化
    x = x - x.mean()
    
    # 计算自相关
    ref = np.dot(x, x) + 1e-9
    ac = np.correlate(x, x, mode="full")[len(x) - 1:]
    ac = ac[: lag_max + 1] / ref
    
    # 在pitch范围内找峰值
    seg = ac[lag_min: lag_max + 1]
    peak = int(np.argmax(seg))
    strength = float(seg[peak])
    lag = lag_min + peak
    
    # 计算pitch
    pitch = float(sr / lag)
    return pitch, max(0.0, min(1.0, strength))
```

**实现原理：**
- 基频范围80~400Hz覆盖成人语音
- 归一化自相关消除幅度影响
- voiced_strength表示周期性强度(0~1)
- 结合RMS阈值区分浊音/清音/静音
